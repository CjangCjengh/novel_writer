## ✨ คุณสมบัติ

- 🤖 **ไปป์ไลน์หลายตัวแทน** — 9 ตัวแทนเฉพาะทางรับผิดชอบ: วางแผน, สร้างโลก, ร่างโครงเรื่อง, เขียนบท, ตรวจสอบ, ขัดเกลา, สรุป
- 📋 **เปรียบเทียบแบบร่างโครงเรื่องหลายแบบ** — สร้างโครงเรื่องหลายสไตล์ (ดราม่า/วรรณกรรม/เชิงพาณิชย์) ให้เลือกหรือรวม
- 🔍 **ตรวจสอบคุณภาพแบบขนาน** — 3 ผู้ตรวจขนาน (ความสอดคล้อง/สไตล์/การปูม) วิเคราะห์ทุกบท พร้อมคะแนนความมั่นใจ
- ✨ **วงรอบขัดเกลาอัตโนมัติ** — ประเมินบทอัตโนมัติและปรับปรุงซ้ำจนถึงเกณฑ์คุณภาพ
- 📍 **จุดตรวจระหว่างเล่ม** — หยุดพักระหว่างเล่มเพื่อตรวจสอบความคืบหน้าและปรับทิศทาง
- 📊 **รายงานสรุปสุดท้าย** — สร้างรายงานฉบับสมบูรณ์อัตโนมัติหลังจบ: วิเคราะห์พล็อต, อาร์คตัวละคร, สถิติ, เบาะแสภาคต่อ
- 🌐 **รองรับ 11+ ภาษาในตัว** — อังกฤษ, จีนตัวย่อ/ตัวเต็ม, ญี่ปุ่น, เกาหลี, เวียดนาม, ไทย, จีนคลาสสิก, ญี่ปุ่นคลาสสิก, ละติน, สันสกฤต
- ➕ **ขยาย i18n ได้ด้วยตัวเอง** — เพิ่มภาษาใหม่ได้ขณะรัน!
- 🔌 **แบ็กเอนด์ที่เข้ากันได้กับ OpenAI** — รองรับ API ที่เข้ากันได้กับ OpenAI
- ⚡ **ทำงานขนาน** — สร้างโลก, คู่มือสไตล์, แบบร่างโครงเรื่อง, ตรวจสอบคุณภาพ ทำงานพร้อมกันทั้งหมด
- 💾 **กลับมาต่อจากจุดตรวจ** — ตรวจจับบทที่เสร็จแล้วโดยอัตโนมัติ
- 📝 **การติดตามเมตาดาต้าที่สมบูรณ์** — ความคืบหน้า, สรุปบท, ตัวติดตามคำใบ้, สถานะตัวละคร
- 🎨 **พารามิเตอร์ปรับแต่งได้** — ไฮเปอร์พารามิเตอร์ทั้งหมดรวมไว้ใน config; การตั้งค่าส่วนตัว (รวมคีย์ API) อยู่ใน `user_config.json` (git-ignore)

### บทบาทตัวแทน

| ตัวแทน | บทบาท |
|--------|-------|
| **PlannerAgent** | รวบรวมความต้องการนิยายผ่านบทสนทนาหลายรอบ (ประเภท, ธีม, โครงสร้าง, ตัวละคร ฯลฯ) |
| **WorldbuildingAgent** | สร้างฉากโลก, โปรไฟล์ตัวละคร, สถานที่, ไทม์ไลน์, ระบบพลัง/เทคโนโลยี (ขนาน) |
| **StyleGuideAgent** | สร้างคู่มือสไตล์การเขียนตามแผนนิยาย |
| **OutlineAgent** | สร้างโครงเรื่องหลัก (เดี่ยวหรือเปรียบเทียบหลายแบบ) และโครงเรื่องรายเล่ม |
| **WriterAgent** | เขียนเนื้อหาบทตามโครงเรื่อง, คู่มือสไตล์, และบริบทต่อเนื่อง |
| **PostWriteAgent** | วิเคราะห์บทที่เสร็จ, สร้างสรุป, อัปเดตเมตาดาต้าความคืบหน้า/คำใบ้ |
| **QualityReviewerAgent** | 3 ผู้ตรวจขนาน (ความสอดคล้อง/สไตล์/การปูม), พร้อมคะแนนความมั่นใจ |
| **PolishAgent** | ขัดเกลาแบบวนซ้ำ: ประเมินคุณภาพ → ปรับปรุง → ประเมินใหม่ จนถึงเกณฑ์ |
| **FinalSummaryAgent** | รายงานฉบับสมบูรณ์หลังจบนิยาย: สถิติ, วิเคราะห์อาร์คตัวละคร, เบาะแสภาคต่อ |

## 🚀 เริ่มต้นอย่างรวดเร็ว

### 1. ติดตั้งแพ็กเกจที่จำเป็น

```bash
pip install -r requirements.txt
```

### 2. ตั้งค่าไฟล์การตั้งค่า

```bash
cp user_config.example.json user_config.json
```

แก้ไข `user_config.json` กรอกข้อมูลรับรอง API และการตั้งค่าที่ต้องการ (เก็บเฉพาะฟิลด์ที่ต้องการเปลี่ยน)

> ⚠️ `user_config.json` ถูก git-ignore แล้ว อย่า commit คีย์ API!
> ดูฟิลด์ทั้งหมดที่ใช้ได้ใน `user_config.example.json`

#### การตั้งค่าเฉพาะนิยาย (ไม่บังคับ)

แต่ละโปรเจกต์นิยายสามารถมี `novel_config.json` ของตัวเองเพื่อเขียนทับพารามิเตอร์การเขียน:

```bash
cp novel_config.example.json output/ชื่อนิยาย/novel_config.json
```

> **ลำดับการโหลดการตั้งค่า**: `config.py` ค่าเริ่มต้น → `user_config.json` (ส่วนกลาง) → `novel_config.json` (เฉพาะนิยาย)
>
> เพื่อความปลอดภัย คีย์ API ใน `novel_config.json` จะถูก **ข้ามโดยอัตโนมัติ** กรุณาตั้งค่า API ใน `user_config.json` ส่วนกลาง
> ดูฟิลด์ทั้งหมดที่ใช้ได้สำหรับแต่ละนิยายใน `novel_config.example.json`

### 3. รัน

```bash
python main.py
```

ระบบจะนำทางคุณผ่าน:

1. **เลือกภาษา** — เลือกภาษานิยาย (หรือเพิ่มภาษาใหม่!)
2. **ตั้งค่าโปรเจกต์** — ตั้งชื่อโปรเจกต์นิยาย
3. **เลือกการดำเนินการ** — ไปป์ไลน์เต็ม, ดำเนินการต่อ, หรือรันแต่ละขั้นตอน

### โหมดการดำเนินการ

| โหมด | คำอธิบาย |
|------|----------|
| **ไปป์ไลน์เต็ม** | ขั้นตอนทั้งหมดจากวางแผนถึงบทสุดท้าย |
| **ดำเนินการต่อ** | ดำเนินต่อจากจุดตรวจล่าสุด |
| **วางแผนเท่านั้น** | บทสนทนาโต้ตอบเพื่อออกแบบนิยาย |
| **สร้างโลกเท่านั้น** | สร้างฉากโลกและคู่มือสไตล์ |
| **โครงเรื่องเท่านั้น** | สร้างโครงเรื่องหลักและรายเล่ม |
| **เขียนเท่านั้น** | เขียนบทในช่วงที่กำหนด |
| **โหมดสร้างหลายเรื่อง** | สร้างนิยายหลายเรื่องตามลำดับ |

## 🌐 เพิ่มภาษาใหม่

เลือก `➕ เพิ่มภาษาใหม่` จากเมนูภาษา ป้อนข้อมูลภาษา แล้ว LLM จะแปลสตริง UI, พรอมต์, และเทมเพลตทั้งหมดโดยอัตโนมัติ สร้างไฟล์ภาษาใหม่ได้ทันที

คุณยังสามารถวางไฟล์ `.py` ในโฟลเดอร์ `locales/` ด้วยตนเอง — ระบบจะตรวจพบอัตโนมัติเมื่อเริ่มต้นครั้งถัดไป

| `ENABLE_FINAL_SUMMARY` | True | สร้างรายงานฉบับสมบูรณ์หลังจบนิยาย |
