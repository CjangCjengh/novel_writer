## ✨ よきところ

- 🤖 **九人のはかりびとの連なり** — おのおの司るところあり：はかりごと、世界の営み、綱目、執筆、審き、潤ひ、総め
- 🌐 **十一あまりの言の葉を内に備ふ** — 英語、唐土の文字、大和言葉、韓の語、越の語、暹羅の語、漢文、文語、羅甸語、梵語
- ➕ **みづから言の葉を増やし得** — 動かしつつ新しき言の葉を加ふること叶ふ。からくりみづから訳して新しき文を生ず
- 🔌 **OpenAIに通ず** — OpenAIと心通はすあらゆる口を用ゐ得
- 📋 **多き綱の比べ** — 数かたち（劇力の型/文深の型/商ひの型）にて綱を立て、択びまたは合はす
- 🔍 **竝べて審く** — 三人の審き手（前後の一貫/文の風/伏線）竝びて各章を見る。信の度あり
- ✨ **潤ひの繰り返し** — 章をみづから評し改め、品の閾に達するまで
- 📍 **巻の間の検め** — 巻の間に歇みて進みを見、向きを正す
- 📊 **了りの総め** — 全書了りて後、報告をみづから成す：筋、人物の弧、数へ、続きの種
- ⚡ **竝びて営む** — 世界、文の風、綱の案、審き、みな竝びて営む
- 💾 **途絶えしところより続く** — 書き終へたる章をおのづから見いだし、絶えしところより再び始む
- 📝 **さまざまの覚え書き** — はかどり、章の要、伏線の行方、人物の様子、みなMarkdownにて記す
- 🎨 **さだめを変へ得** — もろもろの値をconfigに集む。私めたる鍵を含む私設は`user_config.json`に別に置く（git-ignore済み）

### はかりびとの務め

| はかりびと | 務め |
|-----------|------|
| **PlannerAgent** | 幾たびもの語らひにて物語の望みを聞き集む |
| **WorldbuildingAgent** | 世界の設ひ、人物の姿、所々、時の流れ、力の仕組みを造る（竝びて） |
| **StyleGuideAgent** | 方略に基づきて文の風の手引きを造る |
| **OutlineAgent** | 総綱目と巻ごとの綱目を造る |
| **WriterAgent** | 綱目と文の風に従ひて章の文を綴る |
| **PostWriteAgent** | 書き上げたる章を読み解き、要を記す |
| **QualityReviewerAgent** | 三人竝びて審く（一貫/文の風/伏線）。信の度附く |
| **PolishAgent** | みづから繰り返し潤ふ：品を量り→改め→再び量る。閾に至るまで |
| **FinalSummaryAgent** | 全書了りて後の報告：数へ、人物の弧の解き明かし、続きの種 |

## 🚀 はやく始むるには

### 一、要るものを入るる

```bash
pip install -r requirements.txt
```

### 二、秘めたる鍵を設ふ

```bash
cp user_config.example.json user_config.json
```

`user_config.json`を開きて、からくりへの合ひ言葉と好みのさだめを書き入れ給へ。

> ⚠️ `user_config.json`はgit-ignoreなり。秘めたる鍵を決して提出すべからず！
> 用ひ得るすべてのさだめは `user_config.example.json` を見給へ。

#### 物語ごとのさだめ（省くを得）

おのおのの物語に独立の `novel_config.json` を置きて、書きの設ひを覆すを得：

```bash
cp novel_config.example.json output/物語の名/novel_config.json
```

> **さだめの読む順**：`config.py` の常 → `user_config.json`（通の設ひ） → `novel_config.json`（物語ごと）
>
> 安きがために、`novel_config.json` 中のAPI設ひは自ら略さる。APIの設ひは `user_config.json` にて為し給へ。
> 物語ごとに用ひ得る項は `novel_config.example.json` を見給へ。

### 三、動かす

```bash
python main.py
```

かくの如く導かれむ：

1. **言の葉選び** — 物語の言の葉を選ぶ（新しきを加ふることも叶ふ）
2. **著作の設ひ** — 物語に名を付く
3. **なすこと選び** — 通して営む、続きより、もしくはひとつの段のみ

### 営みの種

| 種 | 説き明かし |
|---|---------|
| **通して** | はかりごとより終はりの章まで |
| **続きより** | 絶えしところより続く |
| **はかりごとのみ** | 語らひて物語を思ひ描く |
| **世界の営みのみ** | 世界の設ひと文の風を造る |
| **綱目のみ** | 総綱目と巻ごとの綱目を造る |
| **筆を執るのみ** | 定めたる範の章を綴る |
| **諸著の并び行き** | 複数の物語を順を追ひて紡ぐ |

## 🌐 新しき言の葉を加ふ

言の葉の選びにて`➕ 新しき言の葉を加ふ`を選び、言の葉の名を入るるのみ。からくりみづからすべての文字を訳し、新しき言の葉の文をただちに造る。

`locales/`の内に手づから`.py`の文を置くことも叶ふ。次に動かすをりにおのづから見いだされむ。

## ⚙️ さだめ

| さだめ | 常の値 | 説き明かし |
|--------|--------|---------|
| `TEMPERATURE` | 0.7 | 選びの熱さ |
| `TOP_P` | 0.6 | 核選びの境 |
| `MAX_OUTPUT_TOKENS` | 32768 | 答へごとの印の上限 |
| `CHAPTER_MIN_WORDS` | 3000 | 章の文字の下限 |
| `CHAPTER_MAX_WORDS` | 4000 | 章の文字の上限 |
| `MAX_PARALLEL_WORKERS` | 3 | 竝びて営む数 |
| `MAX_RETRIES` | 5 | 試み直す回数 |

### 高きさだめ

| さだめ | 常の値 | 説き明かし |
|--------|--------|----------|
| `OUTLINE_DRAFT_COUNT` | 3 | 綱の案の数（1にて止む） |
| `ENABLE_PARALLEL_REVIEW` | True | 各章の後に三人竝び審く |
| `ENABLE_POLISH_LOOP` | True | 量り→改めの巡りにて章をみづから潤ふ |
| `POLISH_QUALITY_THRESHOLD` | 8 | 潤ひの巡りを了ふるに要る品の値(1-10) |
| `ENABLE_VOLUME_CHECKPOINT` | True | 巻の間に歇みてはかり問ふ |
| `ENABLE_FINAL_SUMMARY` | True | 全書了りて後に報告を成す |
