## ✨ वैशिष्ट्यानि

- 🤖 **बहु-अभिकर्ता-प्रणाली** — ९ विशिष्टाः अभिकर्तारः प्रत्येकं चरणं निर्वहन्ति: योजना, विश्व-निर्माणम्, रूपरेखा, लेखनम्, समीक्षा, परिष्कारः, तथा च सारांशः
- 🌐 **११+ भाषाः अन्तर्निर्मिताः** — आङ्ग्ल, सरलीकृत/पारम्परिक चीनी, जापानी, कोरियाई, वियतनामी, थाई, शास्त्रीय चीनी, शास्त्रीय जापानी, लैटिन, संस्कृतम्
- ➕ **स्व-विस्तारित-अन्तर्राष्ट्रीयकरणम्** — धावनकाले नूतनभाषा-समर्थनं योजयन्तु! अभिकर्ता स्वस्य LLM उपयुज्य नूतन-स्थान-सञ्चिकाः अनुवादयति जनयति च
- 🔌 **OpenAI-सुसंगत-पृष्ठभागः** — किमपि OpenAI-सुसंगतं API समर्थयति (OpenAI, Azure, तथा च LM Studio, Ollama इत्यादयः स्थानीय-प्रतिनिधयः)
- 📋 **बहुविध-रूपरेखा-तुलना** — अनेकशैल्याम् (नाटकीय/साहित्यिक/वाणिज्यिक) रूपरेखाः जन्यन्ते, चयनाय वा मिश्रणाय वा
- 🔍 **समानान्तर-गुणसमीक्षा** — ३ समानान्तर-समीक्षकाः (सातत्यम्/शैली/पूर्वसूचना) प्रत्येकं अध्यायं विश्लेषयन्ति, विश्वास-अङ्केन सह
- ✨ **परिष्कार-चक्रम्** — अध्यायानि स्वयमेव मूल्यन्ते पुनः पुनः सुधार्यन्ते च, विन्यास्य-गुण-सीमा प्राप्यते यावत्
- 📍 **खण्ड-परीक्षा-बिन्दवः** — खण्डानां मध्ये विरमति, प्रगतिं पश्यति, दिशां समायोजयति
- 📊 **अन्तिम-सारांशः** — कथा-पूर्णतायाः अनन्तरं, पूर्ण-प्रतिवेदनं स्वयमेव जन्यते: कथानक-विश्लेषणम्, पात्र-चापाः, सांख्यिकी, तथा च अनुवर्ती-बीजानि
- ⚡ **समानान्तर-निष्पादनम्** — विश्व-निर्माणम्, शैली-मार्गदर्शिकाः, रूपरेखा-योजनाः, तथा च समीक्षाः सर्वाणि समानान्तरं जन्यन्ते
- 💾 **विरामचिह्न-पुनारम्भः** — पूर्णानि अध्यायानि स्वयमेव अभिजानाति तथा च यत्र स्थगितम् ततः पुनारभते
- 📝 **समृद्ध-मेटादत्ता-अनुवर्तनम्** — प्रगतिः, अध्याय-सारांशाः, पूर्वसूचना/अङ्कुश-अनुवर्तकः, पात्र-स्थितिः — सर्वं Markdown मध्ये रक्षितम्
- 🎨 **अनुकूलनीय-प्राचलाः** — सर्वे अतिप्राचलाः config मध्ये केन्द्रीकृताः; व्यक्तिगत-विन्यासः (API-कुञ्जिकाः सहिताः) पृथक् `user_config.json` मध्ये रक्षिताः (git-ignored)

### अभिकर्ता-भूमिकाः

| अभिकर्ता | उत्तरदायित्वम् |
|----------|----------------|
| **PlannerAgent** | उपन्यास-आवश्यकताः (शैली, विषयः, संरचना, पात्राणि, इत्यादयः) संग्रहीतुं उपयोक्त्रा सह बहु-वारं संवादः |
| **WorldbuildingAgent** | विश्व-सेटिंग्स्, पात्र-प्रोफैल्स्, स्थानानि, समयरेखा, शक्ति/तकनीकी-प्रणालीः जनयति (समानान्तरम्) |
| **StyleGuideAgent** | उपन्यास-योजनायाः आधारं कृत्वा लेखन-शैली-मार्गदर्शिकां रचयति |
| **OutlineAgent** | मुख्य-रूपरेखां तथा च विस्तृत-खण्ड-स्तरीय-रूपरेखाः जनयति |
| **WriterAgent** | रूपरेखाः, शैली-मार्गदर्शिकां, तथा च सातत्य-सन्दर्भं अनुसृत्य अध्याय-पाठं लिखति |
| **PostWriteAgent** | पूर्णानि अध्यायानि विश्लेषणं करोति, सारांशान् जनयति, तथा च प्रगति/पूर्वसूचना-मेटादत्तां अद्यतनं करोति |
| **QualityReviewerAgent** | ३ समानान्तर-समीक्षकाः (सातत्यम्/शैली/पूर्वसूचना), विश्वास-अङ्केन सह |
| **PolishAgent** | स्व-पुनरावृत्ति-परिष्कारः: गुणं मूल्यति → सुधारयति → पुनर्मूल्यति, सीमां प्राप्नोति यावत् |
| **FinalSummaryAgent** | कथा-पूर्णतायाः अनन्तरं पूर्ण-प्रतिवेदनम्: सांख्यिकी, पात्र-चाप-विश्लेषणम्, अनुवर्ती-बीजानि |

## 🚀 शीघ्र-ारम्भः

### 1. निर्भरताः संस्थापयन्तु

```bash
pip install -r requirements.txt
```

### 2. विन्यासं स्थापयन्तु

```bash
cp user_config.example.json user_config.json
```

`user_config.json` सम्पादयन्तु, भवतां API प्रमाणपत्राणि तथा च अभीष्ट-विन्यासान् पूरयन्तु (केवलं परिवर्तनीय-क्षेत्राणि समाविशन्तु)।

> ⚠️ `user_config.json` git-ignored अस्ति। भवतां API कुञ्जिकाः कदापि न समर्पयन्तु!
> सर्वाणि उपलब्ध-क्षेत्राणि `user_config.example.json` मध्ये पश्यन्तु।

#### प्रत्येक-कथा-विन्यासः (ऐच्छिकम्)

प्रत्येकं कथा-परियोजना स्वकीयं `novel_config.json` धारयितुं शक्नोति लेखन-मापदण्डानां अतिलेखनाय:

```bash
cp novel_config.example.json output/कथा_नाम/novel_config.json
```

> **विन्यास-पठन-क्रमः**: `config.py` पूर्वनिर्धारितम् → `user_config.json` (वैश्विकम्) → `novel_config.json` (प्रति-कथा)
>
> सुरक्षार्थं, `novel_config.json` मध्ये API-सम्बन्धि-विन्यासाः **स्वयमेव उपेक्ष्यन्ते**। API-विन्यासं वैश्विके `user_config.json` मध्ये कुर्वन्तु।
> प्रति-कथा उपलब्ध-क्षेत्राणि `novel_config.example.json` मध्ये पश्यन्तु।

### 3. चालयन्तु

```bash
python main.py
```

भवान् एतेषु मार्गदर्शितः भविष्यति:

1. **भाषा-चयनम्** — उपन्यासस्य भाषां चिनोतु (अथवा नूतनां योजयतु!)
2. **परियोजना-स्थापनम्** — उपन्यास-परियोजनायाः नामकरणं करोतु
3. **सञ्चालन-विधिः** — पूर्ण-प्रणाली, पुनारम्भः, अथवा व्यक्तिगत-चरणान् चालयतु

### सञ्चालन-विधयः

| विधिः | विवरणम् |
|-------|---------|
| **पूर्ण-प्रणाली** | योजनातः अन्तिम-अध्याय-पर्यन्तं सम्पूर्ण-प्रवाहं चालयतु |
| **पुनारम्भः** | अन्तिम-विरामचिह्नात् अनुवर्तयतु |
| **केवलं योजना** | उपन्यास-अभिकल्पनायै संवादात्मक-संवादः |
| **केवलं विश्व-निर्माणम्** | विश्व-सेटिंग्स् तथा च शैली-मार्गदर्शिकां जनयतु |
| **केवलं रूपरेखा** | मुख्य- तथा च खण्ड-रूपरेखाः जनयतु |
| **केवलं लेखनम्** | अध्यायानि लिखतु (परिधिं निर्दिशतु) |
| **सामूहिक-विधिः** | अनेक-उपन्यासान् क्रमेण जनयतु |

## 🌐 नूतन-भाषां योजयन्तु

Novel Agent Workflow **स्वयमेव विस्तारयितुं** शक्नोति यत् किमपि नूतन-भाषां समर्थयितुं शक्नोति:

1. भाषा-मेनू तः `➕ नूतन-भाषां योजयन्तु` चिनोतु
2. भाषायाः आङ्ग्ल-नाम, देशीय-नाम, तथा च कोड प्रविशतु
3. LLM स्वयमेव सर्वाणि UI स्ट्रिंग्स्, प्रॉम्प्ट्स्, तथा च टेम्पलेट्स् अनुवादयति
4. नूतन-स्थान-सञ्चिका जन्यते तथा च तत्क्षणमेव उपलब्धा भवति

भवान् `locales/` निर्देशिकायां `.py` स्थान-सञ्चिकां हस्तेन अपि स्थापयितुं शक्नोति — अग्रिम-प्रारम्भे सा स्वयमेव अन्विष्टा भविष्यति।
