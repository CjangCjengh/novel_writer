## ✨ 특징

- 🤖 **아홉 도우미 협업** — 9명의 전문 창작 도우미가 각 단계를 맡음: 구상, 세계관 구축, 줄거리 설계, 집필, 품평, 다듬기, 총괄
- 📋 **여러 초안 줄거리 비교** — 다양한 유형(극적형/문학형/상업형)으로 줄거리를 만들어 선택 또는 통합
- 🔍 **세 방향 동시 품평** — 세 품평가(앞뒤 일관성/문체/복선)가 각 장을 동시에 분석, 확신도 점수 포함
- ✨ **자동 다듬기 반복** — 장을 자동 평가하고 설정한 기준 점수에 도달할 때까지 반복 개선
- 📍 **권 사이 중간 점검** — 권과 권 사이에 잠시 멈춰 진행 상황을 돌아보고 방향 조정
- 📊 **완결 보고서** — 소설 완성 후 전체 보고서를 자동 작성: 줄거리 분석, 인물 성장 곡선, 통계, 후속편 단서
- 🌐 **11개 이상의 내장 언어** — 영어, 간체/번체 중국어, 일본어, 한국어, 베트남어, 태국어, 한문, 고전 일본어, 라틴어, 산스크리트어
- ➕ **자체 확장 다국어** — 실행 중 새 언어 추가 가능!
- 🔌 **OpenAI 호환 연결** — OpenAI 호환 API 지원
- ⚡ **동시 실행** — 세계관 설정, 문체 지침, 줄거리 초안, 품평을 모두 동시 실행
- 💾 **이어쓰기** — 완료된 장을 자동 감지하고 중단된 곳에서 이어쓰기
- 📝 **상세한 창작 기록** — 진행 상황, 장별 요약, 복선 현황, 인물 상태를 Markdown으로 관리
- 🎨 **자유로운 설정** — 모든 설정값을 config에 모아 관리; 개인 설정(API 키 포함)은 `user_config.json`에 분리 (git-ignore 처리됨)

### 도우미 역할

| 도우미 | 역할 |
|--------|------|
| **PlannerAgent** | 여러 차례 대화를 통해 소설 구상 (장르, 주제, 구조, 인물 등) |
| **WorldbuildingAgent** | 세계관, 인물 설정, 장소, 시간 흐름, 힘의 체계 등 동시 작성 |
| **StyleGuideAgent** | 소설 기획에 맞는 문체 지침 작성 |
| **OutlineAgent** | 전체 줄거리(단일 또는 여러 초안 비교) 및 권별 세부 줄거리 작성 |
| **WriterAgent** | 줄거리, 문체 지침, 앞뒤 맥락에 따라 장 본문 집필 |
| **PostWriteAgent** | 완성된 장을 분석하고 요약 작성, 진행/복선 기록 갱신 |
| **QualityReviewerAgent** | 세 방향 동시 품평(일관성/문체/복선), 확신도 점수 포함 |
| **PolishAgent** | 반복 다듬기: 품질 평가 → 개선 → 재평가, 기준 점수 달성까지 |
| **FinalSummaryAgent** | 소설 완성 후 전체 보고서: 통계, 인물 성장 곡선 분석, 후속편 단서 |

## 🚀 빠른 시작

### 1. 의존성 설치

```bash
pip install -r requirements.txt
```

### 2. 설정 파일 구성

```bash
cp user_config.example.json user_config.json
```

`user_config.json`을 편집하여 API 자격 증명과 원하는 설정을 입력하세요 (덮어쓸 항목만 넣으면 됩니다).

> ⚠️ `user_config.json`은 git-ignore 처리되어 있습니다. API 키를 커밋하지 마세요!
> 사용 가능한 전체 항목은 `user_config.example.json`을 참고하세요.

#### 소설별 설정 (선택사항)

각 소설 프로젝트에 독립적인 `novel_config.json`을 배치하여 집필 관련 매개변수를 덮어쓸 수 있습니다:

```bash
cp novel_config.example.json output/소설이름/novel_config.json
```

> **설정 로딩 순서**: `config.py` 기본값 → `user_config.json` (전역) → `novel_config.json` (소설별)
>
> 보안상 `novel_config.json`의 API 관련 설정은 **자동으로 건너뜁니다**. API 설정은 전역 `user_config.json`에서 하세요.
> 사용 가능한 소설별 항목은 `novel_config.example.json`을 참고하세요.

### 3. 실행

```bash
python main.py
```

다음 단계를 안내합니다:

1. **언어 선택** — 소설의 언어 선택 (새 언어 추가도 가능!)
2. **프로젝트 설정** — 소설 프로젝트 이름 지정
3. **작업 선택** — 전체 흐름, 이어쓰기, 또는 개별 단계 실행

### 작업 모드

| 모드 | 설명 |
|------|------|
| **전체 흐름** | 구상부터 마지막 장까지 한번에 |
| **이어쓰기** | 중단된 곳에서 계속 |
| **구상만** | 대화형으로 소설 구상 |
| **세계관 구축만** | 세계관 설정과 문체 지침 작성 |
| **줄거리만** | 전체/권별 줄거리 작성 |
| **집필만** | 지정 범위의 장 집필 |
| **일괄 모드** | 여러 소설을 순서대로 생성 |

## 🌐 새 언어 추가

언어 메뉴에서 `➕ 새 언어 추가`를 선택하고 언어 정보를 입력하면, AI가 모든 화면 문구, 지시문, 서식을 자동 번역하여 새 언어 파일을 바로 만들어 줍니다.

`locales/` 폴더에 `.py` 파일을 직접 넣을 수도 있습니다 — 다음 실행 시 자동으로 인식됩니다.

## ⚙️ 설정

### 기본 설정

| 항목 | 기본값 | 설명 |
|------|--------|------|
| `TEMPERATURE` | 0.7 | 창작 다양성 (높을수록 자유로움) |
| `TOP_P` | 0.6 | 표현 선택 범위 |
| `MAX_OUTPUT_TOKENS` | 32768 | 한 번의 응답에 쓸 수 있는 최대 글자 수 |
| `CHAPTER_MIN_WORDS` | 3000 | 장당 최소 글자 수 |
| `CHAPTER_MAX_WORDS` | 4000 | 장당 최대 글자 수 |
| `MAX_PARALLEL_WORKERS` | 3 | 동시 작업 수 |
| `MAX_RETRIES` | 5 | API 호출 재시도 횟수 |

### 고급 설정

| 항목 | 기본값 | 설명 |
|------|--------|------|
| `OUTLINE_DRAFT_COUNT` | 3 | 줄거리 초안 수 (1로 설정 시 비교 없이 바로 확정) |
| `ENABLE_PARALLEL_REVIEW` | True | 각 장 집필 후 세 방향 동시 품평 |
| `ENABLE_POLISH_LOOP` | True | 평가→개선 반복으로 장을 자동 다듬기 |
| `POLISH_QUALITY_THRESHOLD` | 8 | 다듬기를 멈출 기준 점수 (1-10) |
| `ENABLE_VOLUME_CHECKPOINT` | True | 권 사이에서 잠시 멈춰 확인 |
| `ENABLE_FINAL_SUMMARY` | True | 소설 완성 후 완결 보고서 작성 |
